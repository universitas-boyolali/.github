### ðŸ‘‹ Tentang Kami

Selamat datang di GitHub resmi **Universitas Boyolali (UBY)**. Repositori ini dikelola oleh tim IT kampus berkolaborasi dengan mahasiswa Teknik Informatika dan program studi terkait lainnya.

Tujuan kami adalah:
1.  **Transparansi:** Mengelola sistem informasi kampus yang terbuka dan aman.
2.  **Edukasi:** Menjadi wadah praktek (*sandbox*) bagi mahasiswa untuk belajar standar industri (Git flow, CI/CD, Code Review).
3.  **Inovasi:** Mengembangkan solusi digital untuk masyarakat Boyolali dan sekitarnya.

### ðŸ¤ Panduan Kontribusi (Untuk Mahasiswa)

Ingin kode kamu masuk ke repositori kampus? Ikuti langkah ini:

1.  **Fork** repositori yang ingin kamu kembangkan.
2.  Buat **Branch** baru (`git checkout -b fitur-keren-kamu`).
3.  Lakukan perubahan dan **Commit** (`git commit -m 'Menambahkan fitur keren'`).
4.  **Push** ke branch kamu (`git push origin fitur-keren-kamu`).
5.  Buat **Pull Request** ke repositori ini dan tunggu review dari senior/dosen.

> *"Talk is cheap. Show me the code."* - Linus Torvalds
